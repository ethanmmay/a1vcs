VCS stands for Version Control System which is represented by GitHub. GitHub allows for quick exchange making it efficient and useful.

  1. Create a GitHub Account
  2. Create a new Repository named VCS
  3. Upload VCS files using drag & drop
  4. Open Ubuntu terminal & Install Git
  5. Type "git config --global user.name 'username'"
  6. Type "git config --global user.email 'email@domain.web'"
  7. Go to VCS files on your machine and open terminal
  8. Type "git init" in opened terminal
  9. Type "ll" to see if .git is listed
  10. Type "touch .gitignore" to ignore .git
  11. Type "ll" to see if .gitignore is listed
  12. Show Hidden Files to add "images" and "*.xcf" to .gitignore
  13. Type "git add -A" to track all files with git.
  14. Type "git status" to check tracked files
  15. *If Anything goes Crazy Wrong, type "rm -rf .git" to restart.*
  16. Type "git commit -m 'Initial Commit'" to commit tracked files
  17. Now use "mkdir GitHubClones" to create a remote repo for clones
  18. Then type "git clone https://github.com/CHS-McClendon/VCS_mrm.git ." to clone
